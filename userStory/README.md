# лендинг
Я, как новый пользователь, хочу ознакомиться с предложениями сервиса, чтобы понять, нужно ли мне им пользоваться.

# регистрация
Я, как новый пользователь, хочу иметь возможность зарегистрироваться на платформе, чтобы начать пользоваться ее услугами.

# авторизация
Я, как пользователь, хочу иметь возможность войти в свой аккаунт на платформе, чтобы продолжить пользоваться услугами.

# добавление счета
## добавление карты
- Я, как пользователь, хочу иметь возможность добавить карту из списка предложенных в свои продукты, чтобы использовать ее внутри системы.
### обзорная информация
- Я, как пользователь, хочу при добавлении карты видеть краткое описание и оценку по добавляемой карте, чтобы знать условия карты.
### заполнение кредитной карты
- Я, как пользователь, хочу при добавлении кредитной карты указать текущую задолженность, кредитный лимит, дату выписки и выбрать бонусную программу, чтобы настроить карту внутри системы.
### заполнение дебетовой карты
- Я, как пользователь, хочу при добавлении дебетовой карты указать текущий баланс, дату выписки и выбрать "Ваши повышенные категории*, чтобы настроить карту внутри системы.

## добавление депозита
- Я, как пользователь, хочу иметь возможность добавить депозит в свои продукты, чтобы следить за его состоянием.
### заполнение информации по депозиту
- Я, как пользователь, хочу при добавлении депозита указать баланс и валюту, выбрать цель депозита, чтобы видеть статистику по депозиту в системе.

## добавление наличных
- Я, как пользователь, хочу иметь возможность добавить наличные в свои продукты, чтобы следить за своими средствами.
### заполнение информации по депозиту
- Я, как пользователь, хочу при добавлении наличных указать баланс и валюту, чтобы видеть статистику по наличным в системе.


# перевод средств
- Я, как пользователь, хочу иметь возможность добавить информацию о переводе для любого продукта 


# каталог
- Я, как пользователь, хочу просмотреть сводку по катологу карт, чтобы увидеть все сразу и в одном месте.
## подборки
- Я, как пользователь, хочу видеть подборку карт по каждой сфере жизни: Для здоровья, Пятница и выходные, Карты Мир, Подарки друг другу, Кредитки для ремонта, Чем заправляться, Кэшбэк рублями, Длинный и честный грейс, Карты с доставкой, Универсальные карты, Стабильные условия - чтобы выбрать подходящую карту под конкретное событие.
## популярные карты
- Я, как пользователь, хочу видеть наиболее популярные карты, чтобы выбрать лучшую карту быстрее.
## кредитные карты
- Я, как пользователь, хочу иметь возможность просмотреть список всех кредитных карт с условиями, чтобы точно знать, что для меня подходит лучше.
- Я, как пользователь, хочу получить возможность фильтрации и поиска в списке кредитных карт, потому что точно знаю, какие условия мне точно нужны.
## дебетовые карты
- Я, как пользователь, хочу иметь возможность просмотреть список всех дебетовых карт с условиями, чтобы точно знать, что для меня подходит лучше.
- Я, как пользователь, хочу получить возможность фильтрации и поиска в списке дебетовых карт, потому что точно знаю, какие условия мне точно нужны.
## чем платить по категориям
- Я, как пользователь, хочу чтобы мне подсказывали, где какой картой лучше расплатиться и какой кэшбэк она дает.
- Я, как пользователь, хочу получить подсказку, какие лимиту, оборот и другие условия представляет определенная карта по выбранной категории, чтобы получить максимальные возможности по каждой из них.
